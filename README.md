# Лабораторна робота №4  
## Моделювання цифрового підпису
У межах лабораторної роботи реалізовано просту консольну програму мовою Python, яка демонструє принцип роботи цифрового підпису на основі криптографічної хеш-функції SHA-256.
Програма генерує умовні приватний та публічний ключі з персональних даних, виконує підписання текстового документа та перевіряє коректність підпису. Також продемонстровано, що при зміні документа перевірка підпису повертає некоректний результат, що підтверджує забезпечення цілісності даних.
Технології реалізації: Python, бібліотека hashlib.
